# Google Play Console Image Requirements

Since automatic generation was not possible, please provide the following images in this directory or upload them directly to the Play Console.

## 1. App Icon
*   **Dimensions:** 512 x 512 pixels
*   **Format:** 32-bit PNG (with alpha)
*   **Max file size:** 1 MB

## 2. Feature Graphic
*   **Dimensions:** 1024 x 500 pixels
*   **Format:** JPG or 24-bit PNG (no alpha)
*   **Max file size:** 1 MB

## 3. Phone Screenshots
*   **Required:** At least 2 screenshots
*   **Dimensions:** 
    *   Minimum: 320 px
    *   Maximum: 3840 px
    *   Aspect ratio: 2:1 maximum
    *   *Recommended:* 1080 x 1920 pixels (Portrait)

## 4. 7-inch Tablet Screenshots
*   **Required:** At least 1 screenshot (for "Designed for tablets" list)
*   **Dimensions:**
    *   Minimum: 320 px
    *   Maximum: 3840 px
    *   *Recommended:* 1280 x 800 pixels (Landscape) or 800 x 1280 pixels (Portrait)

## 5. 10-inch Tablet Screenshots
*   **Required:** At least 1 screenshot (for "Designed for tablets" list)
*   **Dimensions:**
    *   Minimum: 320 px
    *   Maximum: 3840 px
    *   *Recommended:* 1920 x 1200 pixels (Landscape) or 1200 x 1920 pixels (Portrait)
