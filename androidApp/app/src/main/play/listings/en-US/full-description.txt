QA Verify and Track is the companion app for developers and QA engineers who want fast visibility into GitHub issues, pull requests, and release readiness. Connect your GitHub repositories to track builds, review PRs, and keep QA moving from your Android device.

Key Features:
- GitHub Integration: Securely connect using your Personal Access Token (PAT).
- Repository Management: View and manage multiple repositories in one place.
- Build Tracking: Keep track of your latest app builds, versions, and platforms (Android/iOS).
- Issue & PR Workflow: Triage issues, review PRs, and post comments without leaving the app.
- PR Actions: Ready, merge, deny, and resolve conflicts with bulk actions.
- Project Visibility: Open your GitHub Project directly inside the app.
- Custom Configuration: Configure tokens per repository for granular access control.
- Secure: Your tokens are stored securely on your device.
